用法一:仅有日期框:
$("#txt1").Calendar();



用法二：带日期和时间框:
$("#txt1").Calendar({ isTime: true });



技巧一：时间范围限制:
start:开始时间（今天）： new Date()
end:结束时间在（在今天的基础上加一年零一个月零一天）： new Date().addYear(1).addMonth(1).addDay(1)
$("#txt3").Calendar({ start: new Date(), end: new Date().addYear(1).addMonth(1).addDay(1) }});

















技巧二：绝对时间范围限制：
$("#txt4").Calendar({ isTime: true, start: "2014-01-01 00:00:00", end: "2016-01-01 00:00:00" });
