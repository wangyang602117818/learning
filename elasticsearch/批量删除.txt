批量删除需要插件 delete-by-query
plugin install delete-by-query
示例:
http://localhost:9200/kosmos-document/permission/_query
{
  "query": {
    "term": {
      "documentId": "57a15a5595180a3dac7e99a1"
    }
  }
}
